# POS SA-Project (sa-conStore)

A Point of Sale (POS) system with beautiful UX/UI based on the provided PDF mockup.

## Features
- Login for staff and owner
- Store all data in local storage (no backend)
- Built with Vue.js, Nuxt.js, and Tailwind CSS

### Staff Login
- Username: staff
- Password: admin

### Owner Login
- Username: admin
- Password: admin

## Tech Stack
- Vue.js
- Nuxt.js
- Tailwind CSS

## Getting Started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```

## Project Structure
- pages/: Main pages (login, dashboard, etc.)
- components/: Reusable UI components
- store/: Local storage logic
- assets/: Styles and images

## Customization
- Update UI based on PDF mockup
- All data is stored in browser local storage

---

This project is designed for demo purposes and does not require a backend or database.
